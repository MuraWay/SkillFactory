                            Проект 7. Возьмёте Бэтмобиль?
            Прогнозирование стоимости автомобиля по характеристикам

Это учебный проект, в котором решается реальная бизнес-задача - определение стоимости автомобиля по его характеристикам.
Датасет - данные о продажах автомобилей с сайта auto.ru

1.

Построена "наивная" модель, точность которой мо метрике MAPE является ориентиром для дальнейших улучшений.
MAPE: 19.88%

Затем:
* Произведен анализ целевой переменной - в модели к ней необходимо применить логарифм
* Ликвидированы пропуски данных (от одного признака с большим количеством пропусков решено отказаться)
* Дубликатов не найдено
* Признаки обработаны, разделены на номинативные, категориальные и бинарные
* Созданы новые признаки
* Выбросы - обрботаны
* Нормализация - применена
* Кодирование - применено
* Выявлены наиболее значимые для целевой переменной признаки
* Корреляционный анализ - проведён

2. 

На подготовленных данных с помощью CatBoost создана модель градиентного бустинга. 
Предварительно применен подбор гиперпараметров (grid_search). 

CatBoostRegressor TEST mape: 11.06%

3. 

Эта же задача решена с помощью DL (модель NN Tabular)

Результаты плавающие. Последний - TEST mape: 11.13%

4. 
Данные содержат текcтовые описания (NLP). 

Текст предобработан:
* очищен, оставлена только кирилица
* проведена лематизация

Наэтих данных создана объединённая Multi-Input нейронная сеть для анализа и табличных данных, и текста одновременно.

TEST mape: 11.28%

5.
Произведена обработка изображений (фото автомобилей)auto.ru
Изображения подвергнуты аугментации. 
данные добавлены в Multi-Input нейронную сеть.



6. 
Проведено ансамблирование градиентного бустинга и нейронной сети.
