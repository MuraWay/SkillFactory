Основные цели и задачи проекта:

Отследить влияние условий жизни учащихся в возрасте от 15 до 22 лет на их успеваемость по математике, 
чтобы на ранней стадии выявлять студентов, находящихся в группе риска.

Информация о данных:
Файл - stud_math.csv
RangeIndex: 395 entries, 0 to 394
Data columns (total 30 columns)

Этапы работы над проектом:

1. Первичный осмотр и обработка данных.
2. Написание функция для обработки данных различного типа.
3. Определение и анализ номинативных переменных: оценка количества уникальных значений, 
    % пропусков в данных, оценка возможности восстановления. Выбор метода восстановления данных
3. Оценка распределения признака для числовых переменных, устранение выбросов.
4. Корреляционный анализ количественных переменных. Отбор и анализ коррелирующих переменных.
5. Анализ номинативных переменных. Отбор значимых для будущей модели.
6. Выводы по заданию
